# Moon Totems WebApp

Currently deployed at [moontotems.com](https://moontotems.com/)

## Apply theme changes

After changing files in `react-app/src/themes`

```bash
yarn theme

# watch
yarn watch-theme
```

## Deploy contracts

```bash
yarn deploy
```

## Verify contracts on etherscan.io

```bash
yarn verify
```
